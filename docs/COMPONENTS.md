# Frontend Components

## Core Components

The `src/components` directory houses reusable UI components.

- `DiscoverySurvey.tsx`: The modal to collect user acquisition sources.
- `BottomNav.tsx`: Bottom navigation for mobile views.
- `ui/`: Standard UI elements (buttons, inputs, etc.) based on standard modern design patterns.

## Contexts

- `contexts/`: React contexts (e.g., Auth, Theme).

## Hooks

- `hooks/useAuth.ts`: Authentication state hook.
- `hooks/useEventStatus.ts`: Event lifecycle state hook.

## State Management

Zustand is used for complex state management (e.g., `cartStore.ts`).
