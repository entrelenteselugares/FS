# Passive Commission Split Model

## Overview

Foto Segundo implements a multi-tier B2B commission model designed to incentivize regional expansion through franchises (Partners). The **Passive Split** ensures that partners earn a percentage of every transaction generated by the professionals they onboard.

## Hierarchy

1. **Master (Matriz)**: Platform owner. Receives the platform fee (approx. 10-15%).
2. **Partner (Franchisee)**: Business owner responsible for a region. Receives a passive commission from all linked professionals.
3. **Professional (Vendedor/Técnico)**: The service provider. Receives the majority of the transaction value.
4. **Editor/Cartório**: Optional secondary tiers for specific service fulfillment.

## Commission Tiers

The split percentages are configured globally in the `PlatformConfig` table:

- `split_matriz`: Platform infrastructure fee.
- `split_captacao`: The professional's share.
- `split_edicao`: Reserved for post-processing technicians.
- `split_cartorio`: Legal/Administrative compliance fee.
- **`split_franchisee`**: The passive B2B commission (e.g., 5-10%).

## Logic Implementation

The logic is centralized in `backend/src/services/pricing.service.ts`:

### Passive Detection

When a sale is processed, the system checks the `ProfessionalNetwork` table:

```typescript
if (options?.professionalId) {
  const network = await prisma.professionalNetwork.findFirst({
    where: { userId: options.professionalId }
  });
  if (network) {
    // Passive Franchisee commission is triggered
    franchisee = +(amount * getPct("split_franchisee")).toFixed(2);
    passiveFranchiseeId = network.partnerId;
  }
}
```

### Direct Sales Mode (Express)

For direct B2B transactions where a professional uses the platform as a tool, the engine applies a **Flat 10% Platform Fee**, leaving the rest to the professional.

## Transparency & Auditing

- Every `Order` record stores a snapshot of the splits at the moment of purchase (`splitMatriz`, `splitCaptacao`, etc.).
- Partners can track their passive earnings in real-time through the Partner Dashboard.
