# Manual do Parceiro Institucional (Cartório / B2B)

Bem-vindo ao painel focado em parcerias passivas (Afiliados Institucionais, Cartórios e Franquias B2B). O sistema trabalha para você.

## Fluxo: Acompanhamento de Indicações e Comissões

1. **Login Institucional:** Acesse a plataforma com a credencial criada pelo administrador e vá para a aba "Parcerias" ou "Afiliados".
   ![Passo 1](../../testes/cartorio/gestao-leads/01-login.png)
2. **Visualização de Leads:** Acompanhe a lista de clientes (ex: noivos, formandos) que agendaram com seu código de Cartório. Você verá o status de cada um (Pendente, Agendado, Finalizado).
   ![Passo 2](../../testes/cartorio/gestao-leads/02-visualizacao-leads.png)
3. **Tabela de Preços e Splits:** Consulte sua tabela personalizada com as margens de lucro pactuadas. Assim que uma transação acontece no cliente atrelado a você, o sistema de Split direciona automaticamente o seu *cut* para o seu Ledger.
   ![Passo 3](../../testes/cartorio/gestao-leads/03-tabela-precos.png)

> **Dica:** O sistema é integrado com a tesouraria. Nenhum desconto aplicado pela matriz afeta a porcentagem de repasse sobre a venda!
