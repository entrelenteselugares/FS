# Milestone v7.0 Requirements: Expansão Total e Go-Live

## 1. Production Operations (OPS)
- [ ] **OPS-01**: Sentry integration for real-time error tracking across frontend and backend.
- [ ] **OPS-02**: Google Analytics 4 (GA4) implementation for tracking conversion funnels (Purchase, Register).
- [ ] **OPS-03**: CI/CD pipeline hardening with automated regression testing before deploy.
- [ ] **OPS-04**: Health monitoring dashboard for Supabase Edge Functions and database load.

## 2. PWA & Mobile Experience (PWA)
- [ ] **PWA-01**: Service Worker implementation for offline asset caching.
- [ ] **PWA-02**: Web Manifest configuration for "Add to Home Screen" support.
- [ ] **PWA-03**: Push Notifications for "New Order" and "Payment Confirmed" alerts.
- [ ] **PWA-04**: Responsive refactor of the selection gallery for high-performance touch interaction.

## 3. Multi-Vertical Diversification (VERT)
- [ ] **VERT-01**: Schema adaptation to support "Escolar" mode (Classrooms, Students, Grades).
- [ ] **VERT-02**: Schema adaptation to support "Esportes" mode (Bib numbers, Athlete search).
- [ ] **VERT-03**: Dynamic Checkout logic for pre-sale (Credit) vs post-sale (Photo selection).
- [ ] **VERT-04**: Administrative control to toggle vertical features per Event.

## 4. Growth & Retention Engine (GROW)
- [ ] **GROW-01**: Dynamic Coupon system (Fixed, Percentage, Expiring).
- [ ] **GROW-02**: Affiliate/Ambassador dashboard for tracking referral sales.
- [ ] **GROW-03**: Automatic retention emails for "Abandoned Cart" or "Gallery Viewed but No Selection".
- [ ] **GROW-04**: Integration with WhatsApp API for automated "Photo Ready" notifications.

---
## Out of Scope (This Milestone)
- Native iOS/Android apps (PWA first).
- Complex stock management for physical gifts (Central Stock focus).
