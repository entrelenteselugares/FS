# Milestone v7.0 Requirements: Expansão Total e Go-Live

_(Archived)_

## 1. Production Operations (OPS)

- [x] **OPS-01**: Sentry integration for real-time error tracking across frontend and backend.
- [x] **OPS-02**: Google Analytics 4 (GA4) implementation for tracking conversion funnels (Purchase, Register).
- [x] **OPS-03**: CI/CD pipeline hardening with automated regression testing before deploy.
- [x] **OPS-04**: Health monitoring dashboard for Supabase Edge Functions and database load.

## 2. PWA & Mobile Experience (PWA)

- [x] **PWA-01**: Service Worker implementation for offline asset caching.
- [x] **PWA-02**: Web Manifest configuration for "Add to Home Screen" support.
- [x] **PWA-03**: Push Notifications for "New Order" and "Payment Confirmed" alerts.
- [x] **PWA-04**: Responsive refactor of the selection gallery for high-performance touch interaction.

## 3. Multi-Vertical Diversification (VERT)

- [x] **VERT-01**: Schema adaptation to support "Escolar" mode (Classrooms, Students, Grades).
- [x] **VERT-02**: Schema adaptation to support "Esportes" mode (Bib numbers, Athlete search).
- [x] **VERT-03**: Dynamic Checkout logic for pre-sale (Credit) vs post-sale (Photo selection).
- [x] **VERT-04**: Administrative control to toggle vertical features per Event.

## 4. Growth & Retention Engine (GROW)

- [x] **GROW-01**: Dynamic Coupon system (Fixed, Percentage, Expiring).
- [x] **GROW-02**: Affiliate/Ambassador dashboard for tracking referral sales.
- [x] **GROW-03**: Automatic retention emails for "Abandoned Cart" or "Gallery Viewed but No Selection".
- [x] **GROW-04**: Integration with WhatsApp API for automated "Photo Ready" notifications.

---

## Out of Scope (This Milestone)

- Native iOS/Android apps (PWA first).
- Complex stock management for physical gifts (Central Stock focus).
