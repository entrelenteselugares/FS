## UI Components Specification

### 1. Ambassador Dashboard
*   **Layout:** Header com resumo (Total Ganhos, Cliques, Conversões).
*   **Charts:** Gráfico de linha simples (Lucide/CSS) mostrando "Cliques nos últimos 7 dias".
*   **Table:** Lista de conversões usando `.lux-card` no mobile e tabela com `.admin-action-btns` no desktop.

### 2. Campaign Manager
*   **Card Style:** `.lux-card` com efeito hover.
*   **Inputs:** Usar `.lux-input` para nomes de campanha e slugs.
*   **Feedback:** Toast de confirmação ao copiar link para o clipboard.

### 3. Checkout Logistics Indicator
*   **Badge:** Pequeno indicador textual no carrinho: "📦 Produção mais próxima: [Nome da Unidade]".
*   **Tooltip:** Explicar que o envio é otimizado para maior velocidade.
